Data to be collected by PIRCS(1c) archive



In the tables below:

    ID = field identification number.
    Root = field's file name root in PIRCS archive.

Note: A table of representative values is available for guidance when preparing output for the archive.

THREE-HOURLY

Model output reported at 00,03,06 ..., 21 UTC. For fluxes (ID 201-208), the output is a three-hour accumulation (e.g. output at 12 UTC is accumulated from 9 UTC- 12 UTC). For mass and temperature (ID 210-216), output is the value at the reporting time. Values are reported on the horizontal grid of the participating model.

All fluxes are positive downward.
