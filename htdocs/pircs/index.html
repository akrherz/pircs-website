FTP NOTES

  • WWW browsers are rarely good ftp clients, for large files or many transfers you may get better performance from a dedicated ftp client program.
  • On UNIX hold the shift key while clicking on a ftp link to get a Save File dialog box.
  • On Windows try the option or alt key while clicking on a ftp link to get a Save File dialog box.
  • On Macintosh try hold the option key while clicking on a ftp link to get a Save File dialog box.
