Help: The System Page

The system page lets you access an OutBox page for each user on the system. You can also see a snapshot image from the system's camera if one is available, and get hardware information about the system.

You can customize the page using an HTML editor. The default editor is Cosmo Create. When editing the page, take care not to remove the comment lines marked DO NOT REMOVE. (Commented, or hidden text appears in Cosmo Create with the icon <!-- -->. If you press Ctrl+E, you'll see the hidden text.)

Use the following buttons on the system page to:

Customize the system's web page.
View a picture from the system's camera.
Display information about this system.
Set up access controls for this system's OutBox pages.
Get help.

Adding a User to Your System Home Page

When a user logs into a new account, an OutBox home page is automatically created for the user, and the user name is added to the OutBox home page for your system.

Removing a User From Your System Home Page

When you remove a user account from the system, the OutBox home page for your system does not automatically remove the user from its user list. To edit the user list, open the file /var/www/htdocs/userList.html and remove the user entry.
