Help: The OutBox Page

The OutBox page contains files and folders that have been published on the Internet by the page's owner. Click a file or folder to open it. Copy a file to your system by using your Web browser's Save Link feature. (In Netscape Navigator, move the mouse over the file, click the right mouse button to see the popup menu, and choose Save Link As.)

Use the following buttons on an OutBox page to:

Go to the system's Web page.
Get help.
