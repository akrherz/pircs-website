The PIRCS collaboration system is using BSCW. Plans are currently underway to add some unique elements to this package related to the PIRCS project. Further information on those plans will be available here when ready.

About BSCW

The BSCW system supports collaboration by providing shared workspaces over the Internet. A shared workspace

allows storage and retrieval of documents and sharing information within a group. This functionality is integrated

with a simple event mechanism to provide each user with an awareness of the activities of others within the

workspace. The system is designed primarily to support self-organising groups.

Find out more about BSCW at http://bscw.gmd.de
