Tape formats supported

Tape and CD formats:
 Directly available at PIRCS are 8mm exabyte and 4mm DDS 2 DAT tapes. Note also that we are able to make CD-ROM's of data. If you are interested in CD-ROM's being made for you please ask.

Also available at IITAP is ability to create older DDS 1 DAT tapes as well as the newer DDS 3 DAT tapes. A need for DDS 3 tapes should be expressed early so that we can make arrangements for this machines use.

Typical commands:

Exabyte Units on DEC Alpha's

Typical commands:
Standard tape command to reading would be:
tar xf /dev/rmt0h

and for writing:
tar cf /dev/rmt0h

Tape to Tape copies:
The system is set up for creating tape copies. To copy from one unit to another you would use a command like:
tar xf /dev/rmt0h | tar cf /dev/rmt1h -

Note the - at the end of the command, this is required to inform tar that it will be obtaining data from a UNIX pipe.
