PIRCS Printers

There are two printers in the PIRCS lab that are on the network. They are listed below along with the printer commands for the various machines.

LaserJet5M
pircs_lj5

DEC UNIX: lpr -Ppircs_lj5

SGI's: lp -dpircs_lj5

Vincent: lpr -Pah3203-lj5
NOTE: while this is fine for PS files, if you are sending text, like source code, the vincent system requires that it be converted before being sent to the printer. This is not a factor with the PIRCS machines but there is nothing I can do about it on Vincent. The system that can be used to print on Vincent is enscript found in the locker gnu. A print string might look something like:
enscript -Pah3203-lj5 file.txt.
There are several nice options in enscript, including uping, line numbering and others that may be of interest to you. For example:
enscript --columns=2 --line-numbers --landscape -Pah3203-lj5 file.txt
Will print the file file.txt in landscape mode, 2 columns per page side by side and put line numbers down.

Mac's: The printers are on the Desktop, or you can use the Chooser under the Apple menu. The printer is in zone Agronomy, as printer pircs_lj5

Windows: The printer discription has been made, you can select it from the print dialog box.
