HDF test pages

We are setting up a test hdf cgi interface system. With this people will be able to come in via the WWW client and view images and data sets from hdf files stored on the system. We will provide a search engine to locate various files.

This is all real ruff right now, when I get the system moving a bit smoother I will polish this up and open it some more to get comments on the usefullness of all this.
