Project Vincent workstations

PIRCS has six Project Vincent machines located in room 3203. Two of these have been expanded to 96 megs. The main server is mounted to:
/local/mnt/pircs

The machine breakdown is:

pv14aa.vincent Dec 3000/300L 32 megs
pv14ab.vincent Dec 3000/300L 32 megs
pv14ac.vincent Dec 3000/300L 96 megs
pv14b3.vincent Dec 3000/300L 96 megs
pv14b4.vincent Dec 3000/300L 32 megs
pv14b5.vincent Dec 3000/300L 32 megs

Three access control lists are used for these machines; They are: pircs_root, pircs_users and pircs_gen. The machines are not general access machines but are rather solely for use by PIRCS staff and researchers. If you are curious about Project Vincent the Iowa State University has provided basic Project Vincent information. That server also provides general information on Iowa State University computers resources.
