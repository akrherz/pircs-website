Cross compiling F90 and C

Chris Anderson, and I resolved a little issue with cross compiling F90 and C. If your main is in C then on the Alpha's you need to tell the F90 compiler that this is the case but you do not need to on the SGI's, a line like:
f90 myf90.f -o myprog myCmain.o
will work fine on the SGI's though it will warn you that you have replaced the default f90 main with your own main (which you of course know, so I don't know why it tells you).
On neither machine did the C compile seem to want to cross compile to the f90 to produce a binary, if people know how to use the C compile to finish a cross compile I would be interested to know and document it here.
