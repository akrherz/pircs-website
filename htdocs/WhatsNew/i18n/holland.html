Netherlands

We provide language and cultural support for the Netherlands.
 
 

Localization

To customize your system to recognize certain language and cultural conventions used in the Netherlands, such as date and time formats, currency format, and sorting order, please follow these steps:

  • Select Desktop Customize Language from the Toolchest to invoke the ipanel (International Settings Control Panel).

    or

  • Type ipanel at a shell prompt.

Country

  • Click the left mouse button over the Country icon at the bottom of the ipanel, then select Holland from the list of countries in the next window.

Language

  • Click the left mouse button over the Language icon at the bottom of the ipanel, then select the language you wish to use from Language Selection window.

Keyboards

Your system has one of several possible keyboards. Although ipanel provides you with a default keyboard configuration, you should either confirm the default or select a different configuration.
 

  • Click the left mouse button over the Keyboard icon at the bottom of the ipanel, then select the keyboard configuration you wish to use from Keyboard Selection window.

Timezone

  • Click the left mouse button. over the Time Zone icon at the bottom of the ipanel, then select the timezone you wish to use from Timezone Selection window.

When you have made all your selections:

  • Click either Set System button or Set Personal button.
  • In the next window, ipanel confirms the new setting you've made.  Click Logout Now button to log out.
  • Log in again with the same account so that these modifications can take effect in any new window.

Notes

The standard keyboard shipped with most SGI systems can enter only ASCII characters by default. Configured as shipped, it does not allow Dutch users to enter or use non-ASCII characters needed for Dutch or other European languages.

The Netherlands location makes it possible to use non-ASCII characters with most system tools. The US ASCII with Compose key keyboard layout is exactly like the default layout for the (101-key) US keyboard, except that the right Alt key (the key immediately to the right of the space bar) behaves as a Compose key, which makes it possible to enter many non-ASCII characters. See the compose(5) and composetable(5) man pages for more information about using the Compose key.
