Österreich

SGI bietet seinen Kunden in Österreich eine landesspezifische Sprachunterstützung.

Die "Desktop"-Meldungen und ein Teil der Online-Dokumentation sowie einige Online-Anwendungen sind in deutscher Sprache verfügbar. Installieren Sie dazu die entsprechende Sprachunterstützung von der separat erhältlichen CD, die die Produktfamilie WorldView(TM) enthält, mit der Sie die Anpassung an verschiedene Landessprachen vornehmen können.

Anpassung

Um Ihr System so anzupassen, daß die für Österreich zutreffenden Konventionen wie z. B. Datums- und Zeitformat, Währungsformat und Sortierreihenfolge von Ihrem System richtig erkannt werden, gehen Sie bitte folgendermaßen vor:

  • Wählen Sie aus den "Werkzeugen" die Option "Desktop -> Anpassen -> Sprache", um ipanel (Bedienfeld für länderspezifische Einstellungen) aufzurufen,

    oder

  • geben Sie an der Shell-Eingabeaufforderung ipanel ein.

Land

  • Klicken Sie mit der linken Maustaste auf das Ländersymbol am unteren Rand von ipanel. Wählen Sie anschließend Österreich aus der Länderliste im nächsten Fenster aus.

Sprache

  • Klicken Sie mit der linken Maustaste auf das Sprachensymbol am unteren Rand von ipanel. Wählen Sie anschließend die gewünschte Sprache im entsprechenden Fenster aus.

Tastaturen

Ihr System ist mit verschiedenen möglichen Tastaturtypen ausgestattet. ipanel bietet Ihnen zwar eine Standard-Tastaturkonfiguration, Sie sollten jedoch diese Standardeinstellung bestätigen oder eine andere Konfiguration auswählen.
 

  • Klicken Sie mit der linken Maustaste auf das Tastatursymbol am unteren Rand von ipanel. Wählen Sie anschließend die gewünschte Tastaturkonfiguration aus der Tastaturliste im entsprechenden Fenster aus.

Zeitzone

  • Klicken Sie mit der linken Maustaste auf das Zeitzonensymbol am unteren Rand von ipanel. Wählen Sie anschließend die gewünschte Zeitzone aus der Zeitzonenliste im entsprechenden Fenster aus.

Nachdem Sie alle Einstellungen vorgenommen haben, gehen Sie folgendermaßen vor:

  • Klicken Sie entweder auf die Schaltfläche "Systemeinstellungen" oder die Schaltfläche "Benutzereinstellungen".
  • Im nächsten Fenster bestätigt ipanel die von Ihnen vorgenommenen neuen Einstellungen.  Klicken Sie anschließend auf die Schaltfläche "Jetzt abmelden", um sich vom System abzumelden.
  • Melden Sie sich erneut mit demselben Konto an, damit Ihre Änderungen für jedes neue Fenster wirksam werden.

Hinweise

Mit der Standardtastatur, mit der die meisten SGI-Systeme ausgeliefert werden, können nur Zeichen eingegeben werden, die im ASCII-Zeichensatz enthalten sind. In der standardmäßig ausgelieferten Konfiguration können deutschsprachige Benutzer keine Zeichen eingeben, die für andere europäische Sprachen benötigt werden, aber nicht im ASCII-Zeichensatz enthalten sind. Wenn Sie als Standort "Österreich (Austria)" angegeben haben, können Sie bei den meisten System-Tools auch auf Zeichen zurückgreifen, die nicht im ASCII-Zeichensatz enthalten sind.

Das Tastatur-Layout "INTL" entspricht bis auf eine Ausnahme exakt dem Standard-Layout der US-amerikanischen Tastatur mit 101 Tasten. Die Ausnahme besteht darin, daß die rechte Alt-Taste (d. h. die Taste rechts neben der Leertaste) als Kombinationstaste ausgelegt ist, sodaß viele nicht im ASCII-Zeichensatz enthaltene Zeichen eingegeben werden können. Weitere Informationen zur Verwendung der Kombinationstaste finden Sie auf den Man Pages zu compose(5) und composetable(5).

Viele Systeme werden mit einer speziell für die deutsche Sprache ausgelegten Tastatur ausgeliefert. Wenn Sie eine deutsche Tastatur besitzen, sollten Sie im Bedienfeld für Spracheinstellungen die Tastatur "DE" auswählen; dadurch werden mit der Tastatur die Zeichen erzeugt, die auf den Tasten abgebildet sind. Mit der rechten Alt-Taste auf der Tastatur "DE" können bei Tasten mit Mehrfachbelegung die zugeordneten Zeichen ausgewählt werden. Die Funktion "Erstellen" wird auf der Tastatur "DE" (wie auf der Man Page zu compose(5) beschrieben) aktiviert, indem Sie die Umschalttaste und die rechte Alt-Taste drücken und beide Tasten gleichzeitig loslassen.

Wenn Sie sich nicht ganz sicher sind, welche Tastaturausführung Sie besitzen, sehen Sie sich zuerst die Eingabetaste an. Wenn die Eingabetaste eine gleichförmig rechteckige Form hat, besitzen Sie eine amerikanische Tastatur. In diesem Fall sollten Sie das Tastatur-Layout "INTL" verwenden. Weist die Eingabetaste keine rechteckige Form auf (d. h. ist die Taste oben etwas breiter geformt als unten), verwenden Sie wahrscheinlich eine deutsche Tastatur.

Wenn Sie aufgrund der Form der Eingabetaste zwar ausschließen können, daß Sie eine amerikanische Tastatur verwenden, bei der Auswahl des Tastatur-Layouts "DE" aber trotzdem nicht die richtigen Zeichen ausgegeben werden, handelt es sich möglicherweise um eine Tastatur für eine andere europäische Sprache. Sie können in diesem Fall probeweise die verschiedenen im Bedienfeld für Spracheinstellungen aufgeführten Tastatur-Layouts einstellen, um die richtige Einstellung für die verwendete Tastatur herauszufinden. Wenden Sie sich an Ihr Vertriebs- und Kundendienstbüro, wenn Sie weitere Informationen oder Hilfe benötigen.
