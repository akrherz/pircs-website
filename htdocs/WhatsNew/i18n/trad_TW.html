Traditional Chinese

We provide language and cultural support for two encodings of Chinese, a simplified encoding for the Chinese mainland and a traditional encoding for Taiwan. The large number of characters in the Chinese language makes it impossible to arrange them on a keyboard. Therefore, an input method, called the input server, is used to make sequences of keyboard events that correspond to each Chinese character. The input server for WorldView(TM) Chinese is called xwnmo(1). It translates your input of phonetic symbols into the equivalent character or characters and provides a list of choices when there is any ambiguity.
 

Localization

To customize your system to recognize certain conventions, such as date and time formats, currency format, and sorting order, as well as to display information and accept input in Chinese, please follow the steps below.

  • Select Desktop Customize Language from the Toolchest to invoke the ipanel (International Settings Control Panel).

    or

  • Type ipanel at a shell prompt.

Country

  • Click the left mouse button over the Country icon at the bottom of the ipanel, then select Tiawan from the list of countries in the next window.

Language

  • Click the left mouse button over the Language icon at the bottom of the ipanel, then select Traditional Chinese from the list of languages in the next window.

Keyboards

Your system has one of several possible keyboards. Although ipanel provides you with a default keyboard configuration, you should either confirm the default or select a different configuration.
 

  • Click the left mouse button over the Keyboard icon at the bottom of the ipanel, then select the keyboard configuration you wish to use from  Keyboard Selection window.

Timezone

  • Click the left mouse button over the Time Zone icon at the bottom of the ipanel, then select the timezone you wish to use from the list of timezones in the next window.

When you have made all your selections:

  • Click either Set System button or Set Personal button.
  • In the next window, ipanel confirms the new setting you've made.  Click Logout Now button to log out.
  • Log in again with the same account so that these modifications can take effect in any new window.

Printing

To print a Chinese language text file to a PostScript compatible printer, use the following command:

% lpcns textfile | lp

where textfile is the name of your file. The lpcns program converts Chinese text to PostScript format. You can print directly from ieditor if your printer supports Chinese characters. If a printer that supports Chinese characters is not available, you may save the document as a text file in ieditor or Showcase before printing it with the lpcns command.
