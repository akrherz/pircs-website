Russia

We provide language and cultural support for Russia.
 
 

Localization

To customize your system to recognize certain conventions used in Russia, such as date and time formats, currency format and sorting order, please follow these steps:

  • Select Desktop Customize Language from the Toolchest to invoke the ipanel (International Settings Control Panel).

    or

  • Type ipanel at a shell prompt.

Country

  • Click the left mouse button over the Country icon at the bottom of the ipanel, then select Russia from the list of countries in the next window.

Language

  • Click the left mouse button over the Language icon at the bottom of the ipanel, then select the language you wish to use from Language Selection window.

Keyboards

Your system has one of several possible keyboards. Although ipanel provides you with a default keyboard configuration, you should either confirm the default or select a different configuration.
 

  • Click the left mouse button over the Keyboard icon at the bottom of the ipanel, then select the keyboard configuration you wish to use from Keyboard Selection window.

Timezone

  • Click the left mouse button. over the Time Zone icon at the bottom of the ipanel, then select the timezone you wish to use from Timezone Selection window.

When you have made all your selections:

  • Click either Set System button or Set Personal button.
  • In the next window, ipanel confirms the new setting you've made.  Click Logout Now button to log out.
  • Log in again with the same account so that these modifications can take effect in any new window.

Support

For further assistance with language and cultural customization questions please contact your local SGI representative.
