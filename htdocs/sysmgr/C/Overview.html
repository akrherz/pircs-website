The System Manager provides access to the tasks that help you set up and administer your workstation. The tasks are organized into the categories described below.

To view a category, click on it in the column at left.

Overview
  Display this overview document.

About This System
  Display information about the configuration of this system.

Search
  Use keywords to search for a task.

Software
  List the tasks that allow you to install software and obtain software licenses.

Hardware and Devices
  List the tasks that allow you to add or remove devices on your workstation. Devices include modems, MIDI devices, printers, mouse pointers, and tablets.

Security and Access Control
  List the tasks that allow you to add and remove user accounts and set the level of security on this workstation.

Networking and Connectivity
  List the tasks that allow you to set up connections to the local network and the Internet.

Files and Data
  List the tasks that allow you to backup and restore the data on this workstation.

System Performance
  List the tasks that allow you to monitor and tune system performance.
